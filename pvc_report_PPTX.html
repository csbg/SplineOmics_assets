This HTML report contains plots visualizing the results from the PVC test.
Right-click on any plot in this report to save it as a .svg (vector graphic) file!

The PVC test is a compound contrast test carried out with limma .
It tests if a given timepoint T is significantly higher or lower than both neighbor timepoints (that is why it is NOT carried out for the first and last timepoint). Tests whether 2T T - T 0 (significantly).
